# Colored Redstone Reborn

Adds various color variants of redstone components.

## Поддерживаемые версии

- **1.21.1 (LTS)**: Стабильная версия с долгосрочной поддержкой.
- **1.21.4 – 1.21.11**: Текущая активная ветка. Используется единый JAR-файл для всех версий в этом диапазоне.

## Установка

1. Установите [NeoForge](https://neoforged.net/).
2. Скачайте мод и поместите его в папку `mods`.
