# Colored Redstone Reborn
**Colored Redstone Reborn** is a mod that adds 16 color variants for basic redstone components. At the moment this mod includes:
- Colored Redstone Lamps
- Colored Redstone Blocks
- Colored Redstone Dust

### Crafting
All colored items can be created in crafting table using other colored components or by combining existing components with a specific dye. Use [**JEI**](https://modrinth.com/mod/jei) to see all recipes in-game.
